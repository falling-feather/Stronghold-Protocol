## 🐛 发现并修复的问题

### 1. Linter 错误（已修复）

#### 问题描述
- **内联样式警告**：HTML 中有多处内联样式（7处）
- **浏览器兼容性错误**：
  - `user-select` 缺少 `-webkit-` 前缀
  - `backdrop-filter` 缺少 `-webkit-` 前缀
  - `viewport` meta 标签包含不推荐属性

#### 修复方案
- 将所有内联样式提取到 `src/styles/main.css`
- 添加浏览器前缀（Safari 兼容）
- 移除 `viewport` 中的 `maximum-scale` 和 `user-scalable`

#### 修复文件
- `index.html` - 移除内联样式，添加 CSS 链接
- `src/styles/main.css` - 新建样式文件

---

### 2. 类型定义缺失（已修复）

#### 问题描述
- `OperatorTemplate` 接口中缺少 `shopLevel` 字段
- 代码中使用了 `template.shopLevel`，但类型定义中没有

#### 修复方案
在 `src/types/index.ts` 中添加：
```typescript
export interface OperatorTemplate {
  // ... 其他字段
  shopLevel: number; // 商店等级（用于临时商店系统）
}
```

#### 修复文件
- `src/types/index.ts`
